# Micro Pico LED Module

Import the LED module into the **main.py**

```python
from micro_pico import led
```

## Functions

Toggle LED (boolean)

```python
led.toggle()
```

Turn LED on

```python
led.on()
```

Turn LED off

```python
led.off()
```

Blink the LED

```python
led.blink("blink ammount int", "duration of pauses in blink")
```
